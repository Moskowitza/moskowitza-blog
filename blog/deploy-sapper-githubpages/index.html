I straight up followed this tutorial. When setting the secret, I selected all options under repo.

I'm using github actions for a few reasons

  • I want to learn more about Github actions to CI and CD
  • Each time I push code to master, deployment will be handled by the action. That's less work for me.
  • I don't want to rely on CircleCI or Jenkins
